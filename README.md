# loop
browser game 'loop'



## development memo

first start :  
install typescript
`npm install`

server start :  
`npm run dev`

build:
`npm run build`

access:  
`http://localhost:3000/`

server stop :  
`Ctrl + C`


static check :
`npm run fix:eslint`
`npx eslint filename (optional:--fix)`
